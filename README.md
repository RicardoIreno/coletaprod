# Prodmais 
Extração e navegação de registros para a Coleta de Produção Científica das Instituições. 

Fontes possíveis: 

+ Base Lattes
+ Web of Science
+ CrossRef (DOI)

## Dependências

1. Elasticsearch 7.4 ou superior
2. PHP 7.4 ou superior
* Dependências do PHP: php-cgi | php-curl | php-xml

## Instalação 

Ver INSTALL.md
